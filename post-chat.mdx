Send a request to an AI character

### Authorizations

<ParamField header="Authorization" type="string" required>
  The Authorization header expects a **Bearer token**. Use your AICord API key (prefixed with `sk-aicord-`). This is a secret key that authenticates your requests. Generate one on the API keys section in account settings.
</ParamField>

### Body

<ParamField body="character" type="string" required>
  The ID of one of your characters. You can get this by editing your character and copying from url:\
  https://aicordapp.com/dashboard/characters/<kbd>8f4295c6-307a-4388-99eb-ab4e17bf01ff</kbd>/edit#basic
</ParamField>

<ParamField default="fff" type="string">
  Enter a description
</ParamField>

<RequestExample>

```python Request
import requests
import json

url = "http://localhost:3000/api/v1/chat"

payload = json.dumps({
  "character": "8f4295c6-307a-4388-99eb-ab4e17bf01ff",
  "messages": [
    {
      "role": "user",
      "content": [
        {
          "type": "text",
          "text": "Hey Cora! Do you know how to make an Applet in AICord?"
        }
      ]
    }
  ]
})
headers = {
  'Content-Type': 'application/json',
  'Authorization': 'Bearer sk-aicord-**************'
}

response = requests.request("POST", url, headers=headers, data=payload)

print(response.text)
```

</RequestExample>
