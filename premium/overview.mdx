## What's included in Premium

- **Infinity tokens** (Unlimited and Supercool plans) — no usage limits on any AICord feature, including image generation
- **More character slots** — create more AI characters per server
- **Access to more capable AI models** — Llama 4, GPT-4o, DeepSeek R1 (coming soon)
- **Voices & voice cloning** — default voice presets on Basic; custom voice cloning on Advanced and above
- **Voice calls** — call characters in Discord voice channels or on the AICord dashboard
- **Extended conversation history** — premium servers store up to 50 messages of temporary context, plus persistent long-term memory for remembering users across sessions
- **Character restriction** — limit a character's activity to specific channels in your server
- **Character selfies** — characters can generate AI images of themselves on request
- **God Mode Prompt** — replace AICord's default system prompt with your own for complete behavior control
- **AICord Applets** — small JavaScript tools that give characters agentic capabilities such as fetching live data, calling external APIs, and reacting to Discord events

<Note>
  Some features are only available on specific plan tiers. Voice cloning requires the Advanced plan or higher. Infinity tokens are only available on Unlimited and Supercool plans. Check the [AICord website](https://aicordapp.com) for a full plan comparison.
</Note>
